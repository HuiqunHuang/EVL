# EVL
Python implementation of paper "Modeling Extreme Events in Time Series Prediction", with some modifications, not official codes.

## Environment and Dependencies
- Python 3.6
- Tensorflow-GPU-2.3.0
- Keras 2.7.0
- Pandas 1.1.5
- Scikit-learn 0.23.1
- CUDA 10.1
- CuDNN 7.6
